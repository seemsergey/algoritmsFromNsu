# Алгоритмы из НГУ - Индекс файлов и алгоритмов

Этот репозиторий содержит реализации различных алгоритмов и структур данных на языке C, изучаемых в Новосибирском государственном университете.

## 📁 Подробная структура репозитория

### 📂 `functions/` - Функции и алгоритмы

#### Sorts (Сортировки)
- **BubbleSort.c** - Сортировка пузырьком. Временная сложность O(n²). Простой алгоритм для обучения.
- **InsertSort.c** - Сортировка вставками. Временная сложность O(n²). Хорошо работает на малых массивах.

#### Dynamic Programming (Динамическое программирование)
- **lagushka.c** - Задача про прыгающее животное (DP на массиве)
- **tips.c** - Задача с советами (оптимизация выбора)
- **fiboncahi/main.c** - Вычисление чисел Фибоначчи с мемоизацией
- **расстояние Левенштейна/** - Вычисление расстояния между двумя строками (Edit Distance DP)

#### Huffman Code v1 (Кодирование Хаффмана)
- **src/** - Исходные файлы реализации алгоритма кодирования Хаффмана (сжатие данных)
- **CMakeLists.txt** - Конфигурация сборки проекта
- **in.txt** - Входные данные для тестирования
- **decode.txt** - Выходные декодированные данные

### 📂 `lib/` - Библиотеки

#### Algoritm Bellman Ford (Алгоритм Беллмана-Форда)
- **bellmanFord.c** - Реализация алгоритма поиска кратчайших путей (работает с отрицательными весами)
- **bellmanFord.h** - Заголовочный файл с объявлениями функций

#### CSR (Compressed Sparse Row - Сжатое представление разреженных матриц)
- **csr.c** - Реализация структуры данных CSR для эффективного хранения разреженных матриц
- **csr.h** - Заголовочный файл с описанием функций работы с CSR

### 📂 `About types/` - Структуры данных и типы

#### Bloom Filter (Фильтр Блума)
- **Bloom filter.c** - Вероятностная структура данных для проверки принадлежности элемента множеству. Использует битовый массив и несколько хеш-функций.

#### DSU (Disjoint Set Union - Система непересекающихся множеств)
- **DSU.c** - Реализация структуры данных DSU с операциями Union и Find. Используется с оптимизациями (union by rank, path compression)
- **tasks/** - Практические задачи для использования DSU

#### Lists (Связные списки)
- **Teorhy.c** - Теория и реализация связных списков (вставка, удаление, поиск)
- **Theory2.c** - Дополнительная информация о двусвязных списках

#### Structs (Кастомные структуры)
- **Teorhy.c** - Реализация собственных структур данных на базе struct

#### Enums (Перечисления)
- **Teorhy.c** - Примеры использования перечислений (enum) в C

#### Unions (Объединения)
- **Teorhy.c** - Примеры использования объединений (union) для эффективного использования памяти

#### Stacks & Queues (Стеки и очереди)
- **Queue.c** - Реализация структуры данных очередь (FIFO)
- **QueueOnStack.c** - Реализация очереди на основе стека
- **StackOnList.c** - Реализация стека на основе связного списка (LIFO)
- **TaskAbout/** - Практические задачи со стеками и очередями

#### Trees (Деревья)
- **Theory.c** - Теория и основные операции с деревьями бинарного поиска
- **RBTree.c** - Реализация красно-чёрного дерева (Red-Black Tree) с балансировкой
- **Дерево сумм/** - Реализация дерева отрезков (Segment Tree)
- **обзоды/** - Обзоры различных алгоритмов на деревьях
- **Tasks/** - Практические задачи с деревьями

#### Heap (Приоритетная очередь / Куча)
- **heap.c** - Реализация приоритетной очереди на основе кучи (heap). Используется для эффективного поиска минимума/максимума.

#### HashMap (Хеш-таблица)
- **HashMap.c** - Базовая реализация хеш-таблицы
- **struct.c** - Вспомогательные структуры для HashMap
- **struct.h** - Заголовочный файл с определениями структур
- **task/** - Практические задачи с использованием HashMap

#### Graph (Графы)
- **main.c** - Основной файл с примерами работы с графами

##### Подразделы графов:
- **Алгоритм Форда-Беллмана/** - Алго��итм поиска кратчайших путей (работает с отрицательными весами)
- **Остовной граф(дерево)/** - Алгоритмы поиска остовного дерева (MST) - Крускал, Прим
- **алгоритм Флойда–Уоршелла/** - Алгоритм поиска всех кратчайших путей между всеми вершинами
- **задачка/** - Практические задачи с графами
- **поиск merge base/** - Поиск наименьшего общего предка (LCA) в дереве
- **слеш тп ртп/** - Специальные алгоритмы на графах
- **топологическая сортировка/** - Топологическая сортировка ориентированного ациклического графа (DAG)

### Contest/ClassContest7 - 7-й классный контест
Продвинутый контест по графовым алгоритмам:

| Файл | Описание | Алгоритм | Сложность |
|------|---------|---------|-----------|
| **ReadMe.md** | Полная документация задачи (на русском) | - | - |
| **main.c** | **Minimum Cost Arborescence (Алгоритм Эдмондса)** - найти ориентированное остовное дерево минимальной стоимости | Edmonds' algorithm (Chu-Liu) | O(V*E) или O(V²*E) |

**Описание задачи:** В компании N сотрудников. Нужно организовать иерархию так, чтобы:
- У каждого (кроме директора) был ровно один наставник
- От директора можно было добраться до любого
- Суммарная стоимость связей была минимальна

**Особенности реализации:**
- Использует BFS для проверки достижимости вершин
- Реализует CSR (Compressed Sparse Row) для оптимизации
- Поддерживает графы до N≤200, M≤2000
- Выводит "IMPOSSIBLE" если нет решения

## 🏗️ Структура данных по категориям

### Основные структуры данных (About types)
- ✅ **Массивы и списки** (Lists) - Связные списки, двусвязные списки
- ✅ **Стеки и очереди** (Stack/Queue) - LIFO, FIFO, реализации на массивах и списках
- ✅ **Деревья** (Trees) - BST, Red-Black Trees, Segment Trees, Heap
- ✅ **Графы** (Graph) - Матрица смежности, списки смежности
- ✅ **Хеш-таблицы** (HashMap) - Хеширование, разрешение коллизий
- ✅ **Система непересекающихся множеств** (DSU) - Union-Find с оптимизациями
- ✅ **Фильтр Блума** (Bloom Filter) - Вероятностная структура данных
- ✅ **Приоритетная очередь** (Heap) - Min-heap, Max-heap
- ✅ **Кастомные структуры** (Structs) - Пользовательские типы данных
- ✅ **Перечисления** (Enums) - Типизированные константы
- ✅ **Объединения** (Unions) - Экономия памяти

### Алгоритмы (Functions)
- ✅ **Сортировки** - Bubble Sort (O(n²)), Insert Sort (O(n²))
- ✅ **Динамическое программирование** - Fibonacci, Edit Distance (Levenshtein), оптимизация задач
- ✅ **Кодирование** - Huffman coding/decoding (сжатие данных)

### Графовые алгоритмы
- ✅ **Алгоритм Беллмана-Форда** - Поиск кратчайших путей с отрицательными весами
- ✅ **Алгоритм Флойда-Уоршелла** - Все кратчайшие пути между всеми вершинами
- ✅ **Остовные деревья** (MST) - Алгоритмы Крускала и Прима
- ✅ **Топологическая сортировка** - Сортировка DAG (directed acyclic graph)
- ✅ **Поиск наименьшего общего предка** (LCA) - Lowest Common Ancestor

### Другие техники
- ✅ **Sliding Window** - Окно скольжения для поиска подмассива
- ✅ **Prefix Sum** - Префиксные суммы для быстрых запросов диапазонов
- ✅ **Битовые операции** - Работа с битовыми флагами
- ✅ **Backtracking** - Перебор с отсечением вариантов
- ✅ **DFS/BFS** - Поиск в глубину и в ширину по графам
