# Алгоритмы из НГУ - Индекс файлов и алгоритмов

Этот репозиторий содержит реализации различных алгоритмов и структур данных на языке C, изучаемых в Новосибирском государственном университете.

## 📁 Структура репозитория

### 📂 `functions/` - Функции и алгоритмы

#### Sorts (Сортировки)
- **BubbleSort.c** - Алгоритм сортировки пузырьком
- **InsertSort.c** - Алгоритм сортировки вставками

#### Dynamic Programming (Динамическое программирование)
- **lagushka.c** - Задача про прыгающее животное (динамическое программирование)
- **tips.c** - Задача с советами (динамическое программирование)
- **fiboncahi/main.c** - Вычисление чисел Фибоначчи (динамическое программирование)
- **расстояние Левенштейна/** - Вычисление расстояния Левенштейна между строками

#### Huffman Code v1 (Кодирование Хаффмана)
- **src/** - Исходные файлы реализации алгоритма кодирования Хаффмана
- **CMakeLists.txt** - Конфигурация сборки проекта
- **in.txt** - Входные данные для тестирования
- **decode.txt** - Выходные декодированные данные

### 📂 `lib/` - Библиотеки

#### Algoritm Bellman Ford (Алгоритм Беллмана-Форда)
- **bellmanFord.c** - Реализация алгоритма поиска кратчайших путей
- **bellmanFord.h** - Заголовочный файл с объявлениями функций

#### CSR (Compressed Sparse Row - Сжатое представление разреженных матриц)
- **csr.c** - Реализация структуры данных CSR
- **csr.h** - Заголовочный файл с описанием функций работы с CSR

### 📂 `About types/` - Структуры данных и типы

#### Bloom Filter (Фильтр Блума)
- **Bloom filter.c** - Вероятностная структура данных для проверки принадлежности элемента множеству
- **Bloom filter.exe** - Скомпилированный исполняемый файл

#### DSU (Disjoint Set Union - Система непересекающихся множеств)
- **DSU.c** - Реализация структуры данных DSU с операциями Union и Find
- **tasks/** - Практические задачи для использования DSU

#### Lists (Списки)
- **Teorhy.c** - Теория и реализация связных списков
- **Theory2.c** - Дополнительная информация о списках

#### Stacks & Queues (Стеки и очереди)
- **Queue.c** - Реализация структуры данных очередь
- **QueueOnStack.c** - Реализация очереди на основе стека
- **StackOnList.c** - Реализация стека на основе связного списка
- **TaskAbout/** - Практические задачи со стеками и очередями

#### Trees (Деревья)
- **Theory.c** - Теория и основные операции с деревьями
- **RBTree.c** - Реализация красно-чёрного дерева (Red-Black Tree)
- **Дерево сумм/** - Реализация дерева отрезков (Segment Tree)
- **обзоды/** - Обзоры различных алгоритмов на деревьях
- **Tasks/** - Практические задачи с деревьями

#### HashMap (Хеш-таблица)
- **HashMap.c** - Базовая реализация хеш-таблицы
- **struct.c** - Вспомогательные структуры для HashMap
- **struct.h** - Заголовочный файл с определениями структур
- **task/** - Практические задачи с использованием HashMap

#### Graph (Графы)
- **main.c** - Основной файл с примерами работы с графами

##### Подразделы графов:
- **Алгоритм Форда-Беллмана/** - Алгоритм поиска кратчайших путей
- **Остовной граф(дерево)/** - Алгоритмы поиска остовного дерева (MST)
- **алгоритм Флойда–Уоршелла/** - Алгоритм поиска всех кратчайших путей
- **задачка/** - Практические задачи с графами
- **поиск merge base/** - Поиск наименьшего общего предка
- **слеш тп ртп/** - Специальные алгоритмы на графах
- **топологическая сортировка/** - Топологическая сортировка графа

#### Other Types
- **Structs/** - Кастомные структуры данных
- **enums/** - Перечисления (enums)
- **uniouns/** - Объединения (unions)
- **Куча/** - Реализация приоритетной очереди (Heap)

### 📂 `Contest/` - Решения контестных задач

Различные папки с решениями задач из контестов:
- **first** - Первый контест
- **second** - Второй контест
- **Third** - Третий контест
- **ClassContest7** - 7-й классный контест
- **SIXsevenContest** - Контест 67

- **тетыре** - Четвёртый контест (название на русском)
- **пятый** - Пятый контест (название на русском)
- **2502** - Контест от 25.02

- ✅ Массивы и списки (Lists)
